# DAIC Workflow Quick Reference

## Commands

```
mek: <task>                  # Start task (Discussion)
/superpowers:brainstorm      # Refine requirements
start^:                      # Gather context (Alignment)
/superpowers:write-plan      # Generate plan
yert                         # Approve plan (Implementation)
/superpowers:execute-plan    # Execute with checkpoints
finito                       # Verify, commit (Check)
```

## Pro Tips

- Use brainstorm even if requirements seem clear
- Review plan carefully before `yert` — scope locks after approval
- Trust checkpoints — review at each pause
- Read the summary after `finito` — it's your future context
